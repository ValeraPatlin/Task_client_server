# Task_client_server

=================================================================
   Разработка серверного ПО для работы по сети
=================================================================

Среда разработки: Qt Creator
Версия библиотек Qt: 5.15
ОС: Windows 10 x64, Linux x64

=================================================================

   Задача: Разработать серверное и клиентское ПО с графическим интерфейсом для работы по сети по протоколу TCP/IP

   В ПО должно быть реализовано:

- Возможность настройки номера порта для входящих соединений
- Приём входящих подключений по TCP порту
- Приём произвольных данных от клиента по протоколу TCP/IP
- Обработка принятых данных согласно алгоритма (см. ниже)
- Отправка обработанных данных клиенту
- Логирование работы сервера в текстовый файл

  Алгоритм обработки входных данных от клиента:
  
Клиент передаёт массивы байт порциями произвольной длины. 
Каждый байт типа quint8.
Сервер обрабатывает каждый байт по следующему алгоритму:

- Инвертировать старший и младший биты
- Применить операцию XOR со значением 0xA5

=================================================================
